# AWS EKS Cost Estimator

Simple Bash script to estimate **near real-time costs** for an EKS cluster with GPU nodes.

Perfect for learning projects where you want to stay in control of spend.

## Features
- Calculates cost from cluster creation time
- Separate rates for EKS control plane and GPU Spot instances
- Easy to customize for different instance types

## Usage

```bash
./eks-cost-estimator.sh
